Projects

Graft: Interactive Graph Theory Sandbox

JavaScript, SVG | January 2026 - Present

A Desmos-style web application for building and exploring graphs through an intuitive visual interface. Features real-time computation of graph properties including degree sequences, connectivity, bipartiteness, and chromatic number. Designed for students testing constructions, instructors creating examples, and researchers prototyping conjectures.

r2a: RISC-V Disassembler

Python, RISC-V | Fall 2025

Developed a formal specification for RISC-V instruction decoding using decision-tree optimization. Built a testing pipeline verifying binary equivalence between assembly and machine code.

Epidemiological Transmission Mapping

Python | Fall 2025

Modeled bacterial infection spread as a complete weighted directed graph using genetic (Hamming distance) and epidemiological data. Implemented greedy algorithms to infer a rooted directed minimum spanning tree (RDMST) representing likely transmission pathways.
